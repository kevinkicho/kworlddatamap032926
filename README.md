# 🌍 World Population Density Map

An interactive data visualization exploring global population distribution using open-source tools and free public data.

![World Population Density Map](https://img.shields.io/badge/Data-World%20Bank%202020-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 🌟 Features

- **Choropleth Map**: Visualize population density across 216 countries
- **Interactive Controls**: Switch between density view, city view, or both
- **Major Cities Overlay**: See the world's largest metropolitan areas
- **Data Insights**: Statistics panel showing global averages and extremes
- **Dark Theme**: Easy on the eyes with GitHub-inspired styling

## 🚀 Live Demo

Visit: https://kevinkicho.github.io/kworlddatamap032926

## 📊 Data Sources

- **Population Density**: World Bank API (2020 data)
- **City Data**: Major metropolitan areas worldwide
- **Visualization**: Datamaps + D3.js

## 🛠️ Tech Stack

- **Backend**: Node.js + Express
- **Frontend**: Vanilla HTML/CSS/JS
- **Mapping**: Datamaps, D3.js, TopoJSON
- **Data**: World Bank Open Data API

## 🏃 Running Locally

```bash
# Clone the repository
git clone https://github.com/kevinkicho/kworlddatamap032926.git

# Install dependencies
npm install

# Start the server
npm start

# Open http://localhost:3000 in your browser
```

## 🎯 Purpose

This project aims to help understand human existence through data visualization, fostering empathy and awareness of global population distribution. By making this data accessible and visual, we hope to contribute to a more equitable world where all people are valued for their positive contributions.
